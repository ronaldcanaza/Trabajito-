Comunicación entre Computadores

REDES

introducción

Las redes informáticas son sistemas que permiten conectar computadoras y dispositivos para compartir información, recursos y servicios.

👉 Ejemplo:

•Compartir internet

•Enviar archivos

•Usar impresoras en red

📌 Importancia:

•Facilitan la comunicación

•Permiten el acceso a información global

•Son base de Internet

2. 🧱 Modelo OSI y TCP/IP

Modelo OSI (7 capas)

Es un modelo que explica cómo viaja la información en una red.

•Física → cables, señales

•Enlace → transmisión de datos

•Red → direcciones IP

•Transporte → control de envío

•Sesión → conexión entre equipos

•Presentación → formato de datos

•Aplicación → uso por el usuario

🔹 Modelo TCP/IP (4 capas)

Es el modelo real usado en Internet.

•Acceso a red

•Internet

•Transporte

•Aplicación

Diferencia

✓OSI es teórico (para aprender)

✓TCP/IP es práctico (se usa en la vida real)

3. 📡 Protocolos

Los protocolos son reglas que permiten la comunicación entre dispositivos.

🔹 Ejemplos importantes:

✓HTTP / HTTPS → Navegar en internet

✓FTP → Transferir archivos

✓TCP/IP → Base de Internet

✓DNS → Traduce nombres de páginas (ej: google.com)

4. 🔹 Topologías de Red

Es la forma en que están conectados los dispositivos.

🔸 Tipos:

•Estrella ⭐: todos conectados a un centro (router)

•Bus 🚌: una sola línea

•Anillo 🔵: forma circular

•Malla 🕸️: todos conectados entre sí

5. 🔹 Software de Red y Simuladores

Son programas que permiten configurar, analizar o simular redes.

🧠 Ejemplo importante:

✓Packet Tracer → Simula redes (muy usado en educación)

✔️ Funciones

•Crear redes virtuales

•Configurar routers y switches

•Practicar sin usar equipos reales

6. 🔹 Dispositivos de Red

Son los equipos que permiten la conexión en la red.

🔌 Principales:

✓Router → Conecta a internet

✓Switch → Conecta dispositivos en red local

✓Módem → Da acceso a internet

✓Access Point → Da WiFi

✓Hub → Dispositivo básico (ya casi no se usa)

CONCLUSIÓN

Las redes son fundamentales hoy en día, ya que permiten la comunicación global. Gracias a modelos como OSI y TCP/IP, protocolos, dispositivos y software, podemos conectarnos y compartir información de manera rápida y eficiente.

“Las redes están en todo lo que usamos hoy, desde el celular hasta el internet, y funcionan gracias a reglas, dispositivos y estructuras bien organizadas.”
