SOFTWARE

Parte Lógica del Computador

¿QUE ES EL SOFTWARE?

El software es la parte lógica de una computadora. Está formado por todos los programas, aplicaciones y sistemas que permiten que el hardware funcione y que el usuario pueda realizar diferentes tareas. En pocas palabras, el software le dice al computador qué hacer y cómo hacerlo. ⚙️ Características del software No es físico (no se puede tocar) Se puede actualizar o modificar Es necesario para que el hardware funcione Puede ser gratuito o de pago Se desarrolla mediante programación

TIPOS DE SOFTWARE

1. 🖥️ Software de sistema

Es el que controla el funcionamiento del computador.

EJEMPLOS

• Windows

• Linux

• MacOS

FUNCIONES

• Administración de hardware

• Controlar la memoria

• Permitir que otros programas funcionen

2. SOFTWARE DE de APLICACIÓN

El software de aplicación es el conjunto de programas diseñados para ayudar al usuario a realizar tareas específicas en una computadora o dispositivo móvil. A diferencia del software de sistema, este es el que el usuario utiliza directamente.

📌2.1. Características del software de aplicación

• Esta diseñado para el usuario final.

•Realiza tareas específicas.

•Es facil de usar.

•Puede ser instalado o usado en linea.

•Depende del sistema operativo oara funcionar.

📌2.2 Funciones principales

El software de aplicación permite:

•Crear documentos

•Realizar cálculos

•Navegar por internet

•Comunicarse (mensajes, videollamadas)

•Editar imágenes, videos o audio

• Jugar videojuegos

📌2.3 Tipos de software de aplicación

📝 a) Software de productividad Ayuda en tareas de oficina o estudio. Ejemplos: Word, Excel, PowerPoint

🌐 b) Software de comunicación Permite comunicarse con otras personas. Ejemplos: WhatsApp, Zoom, Gmail

🎮 c) Software de entretenimiento Se usa para diversión. Ejemplos: videojuegos, reproductores de música

🎓 d) Software educativo Ayuda en el aprendizaje. Ejemplos: plataformas virtuales, apps educativas

🎨 e) Software de diseño Permite crear contenido gráfico. Ejemplos: Photoshop, Canva

🌍 f) Software web Se usa a través de internet sin instalarlo. Ejemplos: Google Docs, redes sociales

📌2.4 Ejemplos de software de aplicación en la vida diaria

•Aplicaciones móviles (WhatsApp, TikTok)

•Navegadores web (Chrome, Firefox)

•Programas de oficina (Word, Excel)

•Plataformas educativas

📌2.5 Importa del software de aplicación

El software de aplicación es importante porque:

•Permite realizar tareas cotidianas

•Facilita el estudio y el trabajo

•Mejora la productividad

•Es esencial en la vida moderna

📌2.6 Diferencia con otros tipos de software

•Software de sistema: controla el hardware

•Software de aplicación: lo usa el usuario

•Software de programación: crea programas

📌2.7 Conclusión

El software de aplicación es fundamental porque permite que las personas interactúen con la tecnología de manera sencilla. Sin este tipo de software, no podríamos usar la computadora para actividades diarias.

“El software de aplicación es el que usamos todos los días para estudiar, trabajar y comunicarnos.”

3 SOFTWARE DE PROGRAMACIÓN

📌3.1 ¿ Que es el software de programación

El software de programación es el conjunto de herramientas que permiten a los desarrolladores crear, modificar, probar y mantener programas informáticos (software). En pocas palabras: Es el software que se usa para hacer otros programas.

📌3.2 Características principales

•Permite escribir código en diferentes lenguajes.

•Facilita la detección de errores (debugging).

•Automatiza procesos de desarrollo.

•Mejora la productividad del programador.

•Puede incluir interfaces gráficas o trabajar en consola.

🧩 3. Tipos de software de programación

📝 a) Editores de código

• Visual studio code

•Sublime text

• Notepad

🐞 e) Depuradores (Debuggers)

Ayudan a encontrar errores en el código.

•GDB

•Visual studio debugger

📌 3.4 Lenguajes de programación relacionados

El software de programación trabaja con lenguajes como:

•Python

•Java

•Java Script

📌3.5 Funciones principales

.•Crear programas y aplicaciones

•Traducir código a lenguaje máquina

•Detectar errores

•Optimizar el rendimiento

•Facilitar el trabajo en equipo

3.6 Importancia del software de programación

El software de programación es fundamental porque:

•Permite el desarrollo de aplicaciones (apps, juegos, sistemas)

•Hace posible la innovación tecnológica

•Es la base de la inteligencia artificial y el desarrollo web

•Impulsa la economía digital

4 Ejemplo práctico

👉 Un programador usa:

•Visual studio code para escribir un codigo

•Git para guardar cambios

•Nodejs oara ejecutar programa

📃5 conclusión

El software de programación es una herramienta esencial en la informática moderna. Gracias a él, los desarrolladores pueden crear todo tipo de aplicaciones que usamos diariamente. Sin este tipo de software, no existirían programas, páginas web ni sistemas tecnológicos actuales.
